# 团子也要跑步 - 步数更新工具

一个基于Next.js的步数更新工具，用于同步运动应用中的步数数据。

## 功能特性

- **步数更新**：支持自定义步数，通过API接口直接同步到运动应用
- **多账号支持**：本地存储账号信息，方便快速切换
- **深色模式**：支持浅色/深色主题切换
- **响应式设计**：适配各种设备屏幕
- **联系信息**：提供开发者联系方式和博客链接

## 技术栈

- **前端框架**：Next.js 14.1.0
- **React**：18.2.0
- **TypeScript**：5.3.0
- **样式**：Tailwind CSS 3.4.1
- **API调用**：Axios 1.6.0
- **其他**：CryptoJS、UUID、Lucide React

## 安装部署

### 本地开发

1. 克隆项目
   ```bash
   git clone <repository-url>
   cd mimotion/web
   ```

2. 安装依赖
   ```bash
   npm install
   ```

3. 启动开发服务器
   ```bash
   npm run dev
   ```

4. 访问应用
   打开浏览器访问 http://localhost:3000

### 部署到Vercel

1. 登录Vercel账号
2. 点击"New Project"
3. 选择从GitHub导入项目
4. 选择你的仓库
5. 点击"Deploy"
6. 部署完成后，Vercel会生成一个访问URL

## 使用说明

1. **步数更新**
   - 输入生日（默认验证：0824）
   - 输入账号和密码
   - 设置目标步数（可通过滑块、预设值或随机生成）
   - 点击"动动吧"按钮提交
